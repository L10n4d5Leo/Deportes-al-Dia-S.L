# Deportes al Día SL

![blob:https://logo.com/318e30b6-4692-467a-8753-a7e55ce5ddf7] 
*(reemplaza con una captura real o logo de tu web cuando lo subas)*

Sitio web dedicado a mantenerte **al día** con lo último en el mundo del deporte. Especializado en **Fútbol**, **Baloncesto** y **Béisbol**, con noticias, resultados en vivo, estadísticas, fichajes, lesiones y mucho más.

Sitio web: [https://deportesaldia-sl.com](https://deportesaldia-sl.com) *(cambia por tu URL real)*  
Demo / Versión en producción: [https://tu-sitio.netlify.app](https://tu-sitio.netlify.app) *(si usas Netlify, Vercel, etc.)*

## ✨ Características principales

- Noticias actualizadas diariamente de las principales ligas
- Secciones dedicadas a **Fútbol** ⚽, **Baloncesto** 🏀 y **Béisbol** ⚾
- Resultados en vivo y marcadores (si lo tienes implementado)
- Estadísticas de jugadores y equipos
- Calendario de partidos y próximos encuentros
- Diseño **responsive** (perfecto en móvil, tablet y escritorio)
- Artículos destacados y categorías
- Búsqueda rápida de noticias y equipos

## 🛠 Tecnologías utilizadas

- **Frontend**: HTML5 • CSS3 • JavaScript (vanilla o con framework)
- **Framework** (si aplica): React / Vue / Next.js / Astro / etc.
- **Estilos**: Tailwind CSS / Bootstrap / CSS puro / SCSS
- **Backend** (si tienes): Node.js / PHP / Python / Supabase / Firebase
- **API's deportivas** (si usas): API-Football, SportsData.io, ESPN API, TheSportsDB, etc.
- **Hosting/Despliegue**: Vercel / Netlify / GitHub Pages / Render
